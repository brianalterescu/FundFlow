
# Fund Flow
FundFlow is a personal finance platform that lets users track transactions, manage budgets, view spending insights, and share a public financial profile. It was created as a Senior Capstone Project for Farmingdale State College.

# Features

- Add, edit, and delete transactions

- Category filtering & CSV export

- Budget and paycheck allocation tools

- Real-time charts & spending insights

- Editable user profiles with public URLs

- Secure login with Firebase Authentication

- Mobile-first responsive design
