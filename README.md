
# Fund Flow
FundFlow is an All-in-One Social Finance Tracker that lets users track transactions, manage budgets, view spending insights, and share a public financial profile. It was created as a Senior Capstone Project for Farmingdale State College. The purpose is to track your money, share insights, and grow smarter together. Fund Flow makes finances simple, social, and seamless.

# Features
- Secure registration/login with Firebase Authentication
- Self Profile View
- Ability to search & view other users profiles
- Follow other users
- Block users and hide your profile from them
- Add Single Transactions
- Export user transaction data within a CSV file
- Import user transaction data with a CSV file
- Annual Wrapped Summary
- Create text posts, like text posts, and view text posts from users you follow
- Delete Account & all associated data
- Create spending & income goals
- Notifications
- Profile Customization, such as a profile picture and display name
- Dashboard to overview key information
- Income Forecast Tool
- Edit transactions
- Admin Panel that allows developers to change display names, profile pictures, delete posts, and accounts
